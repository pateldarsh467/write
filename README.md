# Write

A minimal, distraction-free notepad for focused writing. No accounts, no cloud, no tracking — your words stay on your device.

**[Launch Write →](https://write.pateldarsh.com)**

---

## Features

- **Distraction-free** — Clean interface, no toolbars, just you and your words
- **Works offline** — Installable PWA, works without internet
- **Auto-saves** — Every keystroke saved locally to your browser
- **Multiple documents** — Create, switch, and manage multiple notes
- **Markdown shortcuts** — Type `#`, `-`, `**text**` and watch it transform
- **Dark mode** — Easy on the eyes, toggles with your system or manually
- **Typewriter font** — Switch between serif and typewriter styles
- **Export** — Download your work as `.md` files anytime
- **Privacy-first** — Zero data leaves your browser. Ever.

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `⌘/Ctrl + S` | Download as .md |
| `⌘/Ctrl + D` | Toggle dark/light theme |
| `⌘/Ctrl + B` | Bold |
| `⌘/Ctrl + I` | Italic |
| `Tab` | Indent list |
| `Shift + Tab` | Outdent list |
| `Esc` | Close menu/modal |

## Markdown Shortcuts

Type these at the start of a line, then press space:

| Type | Result |
|------|--------|
| `#` | Heading 1 |
| `##` | Heading 2 |
| `###` | Heading 3 |
| `-` or `*` | Bullet list |
| `1.` | Numbered list |
| `>` | Blockquote |
| `---` | Horizontal divider |

Inline formatting (type and it converts):

| Type | Result |
|------|--------|
| `**text**` | **Bold** |
| `*text*` | *Italic* |
| `` `text` `` | `Code` |


## Install as App

### Desktop (Chrome/Edge)
Click the install icon in the address bar, or Menu → "Install Write"

### iOS
Safari → Share → "Add to Home Screen"

### Android
Chrome → Menu → "Add to Home Screen"

## Privacy

- All data stored in `localStorage` — never leaves your browser
- No analytics, no tracking, no cookies
- No external requests (strict Content Security Policy)
- Open source — verify it yourself
